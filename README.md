codesign --force --sign - minddock.app

## 📋 项目任务规划

### 第一阶段：基础框架 (3-4天)

| 任务         | 预估时间 | 说明                                  |
| ------------ | -------- | ------------------------------------- |
| 项目结构搭建 | 0.5天    | ✅ 已完成 (Electron + React + shadcn) |
| 路由与布局   | 0.5天    | 三栏布局：侧边栏 + 列表 + 编辑器      |
| 主题切换     | 0.5天    | ✅ 已完成                             |
| 本地数据存储 | 1天      | SQLite / IndexedDB 存储笔记           |
| 笔记 CRUD    | 1天      | 创建、读取、更新、删除笔记            |

### 第二阶段：核心功能 (5-6天)

| 任务            | 预估时间 | 说明                               |
| --------------- | -------- | ---------------------------------- |
| Markdown 编辑器 | 2天      | 集成 TipTap / Milkdown / MDXEditor |
| 笔记列表组件    | 1天      | 搜索、筛选、排序                   |
| 分类管理        | 1天      | 所有笔记、代码片段等分类           |
| 代码片段高亮    | 1天      | Shiki / Prism 代码高亮             |

### 第三阶段：AI 功能 (4-5天)

| 任务        | 预估时间 | 说明                     |
| ----------- | -------- | ------------------------ |
| AI 接口集成 | 1天      | Gemini / OpenAI API 封装 |
| AI 总结功能 | 1天      | 一键总结笔记内容         |
| 智能润色    | 1天      | 文本优化、专业化改写     |
| 朗读功能    | 1天      | TTS 文字转语音           |

### 第四阶段：体验优化 (2-3天)

| 任务        | 预估时间 | 说明                     |
| ----------- | -------- | ------------------------ |
| 快捷键支持  | 0.5天    | Cmd+N 新建、Cmd+S 保存等 |
| 拖拽排序    | 0.5天    | 笔记列表拖拽             |
| 导入/导出   | 1天      | Markdown 文件导入导出    |
| UI 细节打磨 | 1天      | 动画、过渡效果           |

### 📅 时间线 (约 2-3 周)

```
Week 1: 基础框架 + 核心功能（编辑器、笔记管理）
Week 2: AI 功能集成 + 分类管理
Week 3: 体验优化 + 测试打包
```

### 🚀 优先级

1. **先做** → 笔记 CRUD + Markdown 编辑器（核心价值）
2. **再做** → AI 功能（差异化特性）
3. **最后** → 导入导出、快捷键等（锦上添花）
